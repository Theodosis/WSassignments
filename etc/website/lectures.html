    • Ζωντανό "στήσιμο" των apache, PHP, MySQL σε windows

    Οδηγίες για το στήσιμο του WAMP:

    1. Κατεβάζετε την τελευταία έκδοση του WAMP
    2. Κάνετε την εγκατάσταση κανονικά, και στο τέλος επιλέγετε να ξεκινήσει
    3. Με αριστερό κλικ στο εικονίδιο επιλέγετε: "Put online"
    4. Τοποθετείτε τα αρχεία σας στο C:\wamp\www\ (ή όπου ορίσατε τον αρχικό φάκελο κατά την εγκατάσταση)
    5. Βλέπετε τα αποτελέσματα στη διεύθυνση http://localhost/ ή στην http://127.0.0.1 από τον browser του υπολογιστή σας (είναι ακριβώς το ίδιο)

    Και του LAMP (σε ubuntu ή debian):

    1. Εγκαθιστάτε τον lamp server με την εντολή: 
      apt-get install lamp-server^
      Κατά τη διάρκεια της εγκατάστασης θα πρέπει να ορίσετε τον κωδικό του root user στη MySQL.
    2. Ο φάκελος www βρίσκεται στη θέση /var/www/, όπου και τοποθετείτε τα αρχεία σας.
    3. Μπορείτε να βρείτε το configuration του apache server στη διεύθυνση /etc/apache2/ και της php στο /etc/php5/apache2/
    4. Βλέπετε τα αποτελέσματα στη διεύθυνση http://localhost/ ή στην http://127.0.0.1 από τον browser του υπολογιστή σας (είναι ακριβώς το ίδιο)

    Επίσης σας δίνουμε και το παράδειγμα με τα αρχεία .htaccess εδώ

    Computer Graphics για το Web

    Τα γραφικά με υπολογιστές είναι εικόνες που δημιουργούνται από προγράμματα υπολογιστών. Στον τομέα της επιστήμης των υπολογιστών που ασχολείται με την κατασκευή και γενικότερα την αναπαράσταση και επεξεργασία εικόνας σε υπολογιστή καταπιανόμαστε με πράγματα όπως η κατασκευή βιντεοπαιχνιδιών και ψηφιακής διασκέδασης, η αποδοτική και πιστή εμφάνιση τρισδιάστατων κόσμων, η κατασκευή σαγινευτικών εφέ που συνεπαίρνουν τους παίκτες και πολλά άλλα μαγικά κόλπα.

    Στην ομιλία μας θα παρουσιάσουμε τους τρόπους με τους οποίους μπορούμε να παράγουμε και να εμφανίσουμε δυναμικά 2D και 3D γραφικά για το web. Θα μιλήσουμε για τις τελευταίες εξελίξεις των τεχνολογιών του web όπως το HTML5, το WebGL, το canvas και τη χρήση των πιο πρόσφατων εκδόσεων της Javascript που επεκτείνουν δραματικά τους ορίζοντες του τι είναι εφικτό μέσα σε μία ιστοσελίδα.

    Στις δύο αυτές ώρες θα μάθετε πώς να φτιάχνετε όμορφες τρισδιάστατες σκηνές με κίνηση μέσα σε ένα site, πώς να αναπαραστάτε 3D γραφικά χρησιμοποιώντας απλές δομές δεδομένων στη Javascript και πώς να ζωγραφίζετε αποδοτικά και ρεαλιστικά εφέ σωματιδίων.

    Θα μάθετε, θα παίξετε, θα γελάσετε, και ελπίζουμε να εμπνευστείτε για να δημιουργήσετε.

    Οι ομιλητές

    Ο Πέτρος Αγγελάτος είναι προπτυχιακός φοιτητής Ηλεκτρολόγος Μηχανικός και Μηχανικός Υπολογιστών στο Εθνικό Μετσόβιο Πολυτεχνείο. Έχει το δικό του start-up στην Αθήνα, την Kamibu, που ασχολείται με την ανάπτυξη video games για το web με τη χρήση του WebGL. Στο παρελθόν εργάστηκε στην ομάδα ανάπτυξης του κοινωνικού δικτύο zino.gr που αριθμεί 19,000 μέλη. Τα ερευνητικά του ενδιαφέροντα περιλαμβάνουν τον προγραμματισμό WebGL και shaders αλλά και τις βελτιστοποιήσεις στον τομέα της γραφικής υπολογιστών. Τον χειμώνα του 2010 δίδαξε μαζί με το Διονύση ένα εξαμηνιαίο σεμινάριο με θέμα τις τεχνολογίες ανάπτυξης web εφαρμογών.

    Ο Διονύσης Ζήνδρος είναι προπτυχιακός φοιτητής στην ίδια σχολή. Στο παρελθόν ίδρυσε το zino.gr, ήταν στα ιδρυτικά μέλη της Kamibu, εργάστηκε στην ομάδα ανάπτυξης του κοινωνικού δικτύου deviantART που σήμερα αριθμεί 14 εκατομμύρια μέλη και είναι μεταλλιούχος στη βαλκανική ολυμπιάδα πληροφορικής. Τα ενδιαφέροντά του περιλαμβάνουν τις βάσεις δεδομένων, τους αλγορίθμους, τα γραφικά με υπολογιστές, την ανάπτυξη web εφαρμογών, τα κοινωνικά δίκτυα και τη διδασκαλία.
