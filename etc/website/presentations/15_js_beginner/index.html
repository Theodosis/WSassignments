HTML & JavaScript

  • Ετικέτα <script>
    • Χρησιμοποιείται για να φορτώσει κώδικα JavaScript
  • Ιδιότητα src
    • Έχει τιμή το αρχείο JS που πρέπει να φορτώσει
    • Μπορεί να παραληφθεί
  • Ιδιότητα type
    • Έχει τιμή "text/javascript". Πάντα.
<script type="text/javascript" src="foo.js"></script>

    </head>
    <body>
        <script type="text/javascript" src="foo.js"></script>
    </body>
</html>

Basic Syntax

  • Ο κώδικας σε κάθε script αρχίζει να εκτελείται την ώρα που τον "διαβάζει" ο browser
  • Ο κώδικας εκτελείται σειριακά - η μία εντολή μετά την άλλη
  • Κάθε εντολή τελειώνει σε ;
  • Δε λειτουργεί τυπώνοντας πράγματα (όπως η PHP)

alert()

Functions

  • Παρόμοιες με συναρτήσεις σε C, C++, Java, PHP, ...
  • Ορίζουν υπο-ρουτίνες που κάνουν συγκεκριμένη δουλειά
  • Σύνταξη:
    • Ορίζονται με τη λέξη κλειδί function
    • Ακολουθεί το όνομα της συνάρτησης
    • Ακολουθούν τα ονόματα των ορισμάτων σε () χωρισμένα με κόμα

Return

  • Οι συναρτήσεις επιστρέφουν τιμή με return
  • Η τιμή επιστροφής χρησιμοποιείται όπου έγινε η κλήση
  • Επιστροφή σημαίνει τερματισμός της συνάρτησης
  • Δεν ορίζουμε τύπο επιστροφής
  • Δεν είναι υποχρεωτικό

Functions

  • Οι συναρτήσεις είναι μεταβλητές!!
function foo(){
    alert( 'hello' );
}

var foo = 5;
alert( typeof( foo ) );
foo(); //Σφάλμα. Το 5 δεν είναι συνάρτηση!

Objects

  • Στη JavaScript όλα είναι αντικείμενα!
  • Τι είναι ένα αντικείμενο;
    • Ένα λεξικό: Αντιστοίχιση τιμών σε κλειδιά
  • Οι τιμές του λεξικού μπορεί να είναι
    • Άλλο αντικέιμενο
    • Αριθμός
    • Αλφαριθμητικό
    • Συνάρτηση

Objects

  • Δήλωση αντικειμένου 
    // Κενό αντικείμενο
var a = {};

// αντικείμενο με δύο ιδιότητες
var b = {
    foo: 5,
    bar: 'hello'
};

Objects

  • Διαβάζουμε τις τιμές ενός αντικειμένου με object[ 'key' ]
  • Ή καλύτερα object.key
var a = {
    foo: 'bar'
};
var b = a.foo; // Το b έχει την τιμή 'bar'

Objects

  • Μπορούμε να προσθέσουμε ιδιότητες μετά τη δημιουργία
var a = {
    foo: 0
}
a.bar = 1; //Προσθήκη ιδιότητας bar με τιμή 1

Objects

  • Μέθοδοι αντικειμένων
  • Είναι κλειδιά που η τιμή τους είναι συνάρτηση
var a = {
    foo: function(){
        return "I am a method";
    },
    bar: "I am a property"
};

alert( a.foo );
alert( a.foo() );
alert( a.bar );

Strings

  • Μια τιμή που είναι μια σειρά από αριθμούς, χαρακτήρες, γράμματα, ...
  • Το μήκος μπορεί να αλλάζει
  • Δεν υπάρχει περιορισμός μήκους
  • Δεν πρόκειται για πίνακες από χαρακτήρες
  • Μπαίνουν σε μονά ή διπλά εισαγωγικά

Strings as Objects

  • Κάθε αλφαριθμητικό έχει μεθόδους:
    • search - Αναζήτηση μέσα στο String. Επιστρέφει τη θέση
    • split - Χωρίζει το string σε κομμάτια. Επιστρέφει πίνακα
    • substr - Παίρνει κομμάτι από το string
var a = 'a-b-c-d-e';
var b = a.split( '-' );

alert( b );
alert( b[ 3 ] );

Numbers

  • Όλοι οι αριθμοί είναι τύπου Number
  • 64bit κινητής υποδιαστολής

Arrays

  • Δηλώνονται με []
  • Οι τιμές του χωρίζονται με κόμματα
  • Η αρίθμηση ξεκινάει από το 0
var a = [ 1, 'two', 3, [ 'a', 'b' ] ];
alert( a[ 0 ] );
alert( a[ 1 ] );
alert( a[ 3 ][ 1 ] );

Arrays

  • Έχουν χρήσιμες ιδιότητες και μεθόδους
  • Ιδιότητα length: Περιέχει το μήκος του πίνακα: 
    var a = [ 1, 2, 3, 'a' ];
alert( a.length );
  • Μέθοδοι
    • pop();
    • push();
    • shift();
    • slice();
    • reverse();
    • join();
    • sort();
var a = [ 1, 2, 3, 4 ];
var b = a.pop();
// a: [1,2,3]; 
// b: 4; 
a.unshift( b );
// a: [4,1,2,3]; 
a.push( 40 );
// a: [4,1,2,3,40]; 
a.sort();
// a: [1,2,3,4,40];
