function css()

  • Εκτελείται σε ένα αντικείμενο που έχει επιστραφεί από jQuery
  • Διαφορετική συμπεριφορά ανάλογα με τα ορίσματα!!
    • 1 όρισμα: διαβάζει τη css
    • 2 ορίσματα: αλλάζει τη css
  • Οι κανόνες μέσω αυτής της συνάρτησης ισοδυναμούν με τους inline κανόνες, οπότε έχουν μεγαλύτερη ειδικότητα από όλους

css( property );

Επιστρέφει την css ιδιότητα του πρώτου στοιχείου

</script>

function show( duration );

  • Με ένα όρισμα
    • Εμφανίζει σταδιακά τα στοιχεία στα οποία εφαρμόζεται
    • Η εμφάνιση διαρκεί όσο της πούμε σε milliseconds
    • Η διάρκεια μπορεί να είναι και λέξη, όπως 'fast', 'slow', 'normal'
    • fast = 200ms, normal = 400ms, slow = 600ms
    <p id="answer" style="display: none;">42</p>
42
var a = $( '#answer' );
a.show( 'slow' );

class juggling

  • function addClass( string );
    • Προσθέτει στα στοιχεία τις κλάσεις που της περνάμε, χωρισμένες με κενό
    • Δεν αντικαθιστά τις ήδη υπάρχουσες κλάσεις
  • function removeClass( string );
    • Αφαιρεί από τα στοιχεία τις κλάσεις που της περνάμε, χωρισμένες με κενό
    • Αν δεν υπάρχουν ήδη δεν κάνει τίποτα
  • function toggleClass( string );
    • Αν ένα στοιχείο έχει την κλάση την αφαιρεί
    • Αν ένα στοιχείο δεν έχει την κλάση την προσθέτει

function hasClass( string );

  • Επιστρέφει true αν οποιοδήποτε από τα στοιχεία έχει όλες τις κλάσεις του string
<p class="foo bar"></p>
<p class="bar second"></p>

var a = $( 'p' ).hasClass( 'bar foo' );
var b = $( 'p.second' ).hasClass( 'bar' );
var c = $( 'p.second' ).hasClass( 'bar first' );
// a: true
// b: true
// c: false

function attr( property, value );

Document Traversing

  • Συναρτήσεις διάσχισης εγγράφου
  • Μας πηγαίνουν σε σημεία του εγγράφου με βάση τα επιλεγμένα στοιχεία
  • Επιστρέφουν ένα αντικείμενο jQuery με σύνολο στοιχείων διαφορετικά από αυτά του αρχικού συνόλου
  • Όλες οι συναρτήσεις μπορούν να φιλτραρισθούν με επιλογέα

Descendants

  • children( [selector] )
    • Επιστρέφει τα άμεσα παιδιά των στοιχείων σε jQuery αναπαράσταση.
    • Στο καινούριο σύνολο στοιχείων δεν υπάρχουν οι γονείς
  • find( selector )
    • Επιστρέφει όλους τους απογόνους των στοιχείων σε jQuery αναπαράσταση.
    • Η χρήση επιλογέα είναι υποχρεωτική. Διαφορετικά επιστρέφεται άδειο σύνολο
    • Στο καινούριο σύνολο στοιχείων δεν υπάρχει το αρχικό σύνολο

Siblings

  • siblings( [selector] )
    • Επιστρέφει όλα τα αδέρφια του των στοιχείων, χωρίς τα ίδια τα στοιχεία
    • Αν στο αρχικό σύνολο υπάρχουν 2 αδέρφια, θα παραμείνουν επιλεγμένα (θα επιλεγεί το καθ' ένα ως αδερφός του άλλου)
  • next( [selector] )
    • Επιστρέφει τον επόμενο αδερφό από κάθε στοιχείο του αρχικού συνόλου
  • prev( [selector] )
    • Επιστρέφει τον προηγούμενο αδερφό από κάθε στοιχείο του αρχικού συνόλου

Ancestors

  • parent( [selector] )
    • Επιστρέφει όλους τους πατέρες των στοιχείων
  • closest( selector )
    • Επιστρέφει τον πρώτο πρόγονο, σύμφωνα με τον επιλογέα
    • Ο επιλογέας είναι υποχρεωτικός, αλλιώς το επιστρεφόμενο σύνολο θα είναι κενό

Filter

  • filter( selector )
    • Φιλτράρει το υπάρχον σετ στοιχείων, σύμφωνα με τον επιλογέα
    • επιστρέφει υποσύνολο των προηγούμενων στοιχείων

Asynchronous Javascript And XML (AJAX)

  • Συνηθίζουμε να χρησιμοποιούμε JSON ή κείμενο
  • Επιτρέπει HTTP συνδέσεις μέσω JavaScript: POST και GET
  • Είναι διαφανής προς το χρήστη

Pros & Cons

  • Πλεονεκτήματα
    • Ζωντανές εφαρμογές
    • Φόρτωση νέου περιεχομένου χωρίς refresh
    • Διάβασμα των δεδομένων που θέλουμε απευθείας στην JavaScript
    • Αλλαγή μόνο μέρους μιας εφαρμογής
    • Μεγάλη ταχύτητα απόκρισης (Δεν απαιτείται η ανακατασκευή ολόκληρης της σελίδας)
  • Μειονεκτήματα
    • Το κουμπί "πίσω" δε συμπεριφέρεται πάντα όπως θα περιμέναμε
    • Το κουμπί "bookmark" δε συμπεριφέρεται πάντα όπως θα περιμέναμε
    • Οι περισσότερες αράχνες δεν τρέχουν JavaScript, οπότε ενδέχεται να μην καταγράψουν το περιεχόμενο που παράγεται από τη σελίδα
    • Δε μπορούμε να φορτώσουμε οποιαδήποτε διεύθυνση (Cross Origin Request)

$.get( address, data, callback );

  • Κάνει ένα HTTP GET αίτημα στο address με GET παραμέτρους το data
  • Η διεύθυνση είναι ένα συνηθισμένο url, σχετικό ή απόλυτο
    • ΠΡΟΣΟΧΗ! Η διεύθυνση δε μπορεί να "δείχνει" σε σελίδα εκτός αυτής από την οποία καλείται.
  • Τα δεδομένα είναι ένα απλό αντικείμενο javascript, με τα labels να λειτουργούν ως το όνομα του κάθε πεδίου και την τιμή ως η τιμή
  • Το callback είναι μια συνάρτηση η οποία θα εκτελεστεί μόλις ολοκληρωθεί η λήψη των αποτελεσμάτων.
  • Ώς παράμετρος στη συνάρτηση callback περνιέται το αποτέλεσμα που έστειλε ο server
